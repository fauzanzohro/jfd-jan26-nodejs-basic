github adalah: sebuah website tempat berbagi bermacam macam source code

selain github ada juga 
gitlab,bitbucket,gitkraken

fungsi utama github adalah :

1. untuk menyimpan source code secara online
2. untuk kolaborasi antar developer
3. sebagai alat atau tools untuk menarik source code ke server 
4. untuk versioning / kembali keversi sebelumnya

status dalam git:
U / Untracked => file yang belum terdeteksi oleh sistem git.
M / Modified  => file yg existing di modifikasi atau di lakukan perubahan isi file.
A / Added     => file sudah memasuki tahap add 




langkah langkah  mengirim source code ke github
1. add     =>  keluar rumah
2. commit  => udah di jalan dan udah sampe di kantor
3. push    => masuk ke dalam kantor


langkahlangkah menghubungkan repo baru di lokal & server github
git init => mendaftarkan repo lokal kita sebagai repo  yang sudah memakai sistem git
git branc -M main => mengganti cabang utama dengan nama main
git remote add origin https://link-repo-github-kita => menghubungkan repo lokal dengan repo tujuan
git config --global user.email "youremail@provider.com"
git config --global user.name "yourAccountName"
git add . => menambahkan semua file
git commit -m "pesan commit"
git push -u origin main => finalisasi pengiriman source code
jika push kedua kali dst cukup ketik perintah  git push


syarat kolaborasi
1. Invite developernya ke repo kita
2. narik perubahan dari tim collaborator lain :git pull
